## ✨ 핵심 비주얼 컨셉 3가지 및 디자인 브리프

### 💡 컨셉 1: 마법의 주문 (The Magical Spell)
**🎯 목표:** '소원 실현'이라는 감성적 보상을 극대화하여 즉각적인 환상과 몰입을 유도. (가장 감성적 접근)
**🔗 후킹 전략 연결:** '극단적인 결과 제시'에 초점.

| 항목 | 상세 내용 |
| :--- | :--- |
| **비주얼 테마** | 동화 속 마법 주문 또는 고대 문서의 느낌. 소원을 현실로 만드는 '마법 공식'을 시각화. |
| **컬러 팔레트 (Color Palette)** | **Primary:** `#8A2BE2` (강렬한 보라/마법), **Secondary:** `#FFD700` (황금빛 강조), **Accent:** `#1E1E1E` (깊은 검정, 텍스트 대비) |
| **타이포그래피 (Typography)** | **Header:** Playfair Display Bold (고전적이고 신비로운 느낌), **Body:** Noto Sans KR (가독성 및 현대적인 정리감) |
| **레이아웃 좌표** | 중앙 집중형. 핵심 공식(To 부정사 구조)을 황금 테두리 안에 배치하여 시선이 즉시 집중되도록 함. |
| **핵심 디자인 요소** | 빛의 입자(Sparkle effect), 고대 문자가 새겨진 텍스처 배경, 마법 지팡이나 열쇠 등의 상징적 오브젝트 삽입. |

### 💡 컨셉 2: 시스템 설계도 (The Blueprint of Reality)
**🎯 목표:** 'To 부정사'라는 추상적인 개념을 체계적이고 논리적인 과정으로 제시하여, 지식 욕구를 충족시키고 신뢰감을 형성. (가장 논리적 접근)
**🔗 후킹 전략 연결:** '구조화된 본문 설명'에 초점.

| 항목 | 상세 내용 |
| :--- | :--- |
| **비주얼 테마** | 건축 설계도, 회로 기판(Circuit Board), 또는 깔끔한 데이터 시각화 느낌. 복잡한 과정이 단순한 공식으로 정리되는 모습. |
| **컬러 팔레트 (Color Palette)** | **Primary:** `#007BFF` (신뢰의 파란색), **Secondary:** `#343A40` (짙은 회색, 구조감 부여), **Accent:** `#00FF00` (네온 그린, 데이터 강조) |
| **타이포그래피 (Typography)** | **Header:** Montserrat Bold (견고함과 명확성), **Body:** Pretendard (기술적이고 깔끔한 느낌) |
| **레이아웃 좌표** | 그리드 기반의 분할 레이아웃. 화면을 세 부분(Step 1, Step 2, Result)으로 명확히 나누어 각 단계별 공식을 강조. |
| **핵심 디자인 요소** | 직선적인 라인과 박스 처리, 화살표를 이용한 논리적 흐름 표시. 데이터 블록이나 체크리스트 형식의 시각화. |
